___
#### To run the program, make sure you are in the program's folder, then use the Run Command below.
___
#### Note that maps appear in the form `map[k:v k:v]` when printed with `fmt.Println`.
___
##### Run Command:

`$ go run maps.go`

##### Results:
```
map: map[k1:7 k2:13]
v1:  7
len: 2
map: map[k1:7]
prs: false
map: map[bar:2 foo:1]
```
